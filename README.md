Airbnb console
this console works like the Shell but limited to a specific use-case
1-Create a new object (ex: a new User or a new Place)
2-Retrieve an object from a file, a database etc…
3-Do operations on objects (count, compute stats, etc…)
4-Update attributes of an object
5-Destroy an object
to start the console write the following command (run the program to start)
$ ./console.py
